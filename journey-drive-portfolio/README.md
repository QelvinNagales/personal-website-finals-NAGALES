# Journey Drive Portfolio

An interactive 3D driving portfolio built with React, Three.js, and NestJS.

## Tech Stack

- **Frontend:** React, TypeScript, Vite, React Three Fiber, Tailwind CSS, shadcn/ui
- **Backend:** NestJS, Supabase
- **3D Engine:** Three.js with Catmull-Rom spline road system

## Getting Started

### Prerequisites

- Node.js & npm installed

### Frontend

```sh
npm install
npm run dev
```

### Backend

```sh
cd backend
npm install
npm run start:dev
```

## Deployment

Configured for Vercel deployment. See `vercel.json` for routing configuration.
